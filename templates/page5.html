5. 분석 결과 시각화

최근 쇼츠, 릴스 등 짧은 영상 유행으로 인한 드라마 시간 분석

  • 드라마 시간을 분으로 변환
  • 연도별로 그룹화하고 시간의 평균을 시각화 하여 분석

분석 결과

  • 트렌드를 따라 드라마 시간도 감소할거라는 추측과 달리 큰 차이가 나지 않는다
  • 어떤 이유인지 다른 컬럼을 통해 확인

시간 대신 회차가 감소하는지 분석

  • 연도별로 그룹화하고 회차의 평균을 시각화 하여 분석

분석 결과

  • 감소하는 그래프 확인
  • 2003년 ~ 2010년 그래프가 이상

드라마 회차 이상값 확인

  • 해당연도들을 인덱싱하여 해당 값 확인

분석 결과

  • 2003년 ~ 2010년은 모두 데이터가 1개
  • 객관적인 데이터라고 볼 수 없다.

데이터가 비교적 많은 최근 8년 동안의 데이터만을 활용하여 시각화

분석 결과

  • 감소하는 그래프 확인
  • 쇼츠, 릴스 등 짧은 영상이 트렌드인 지금 드라마 시간이 줄어들지 않았지만 회차는 줄어든 모습을 보인다.

최근 "오징어 게임" 등 연령제한 K드라마 증가

  • 연도별 연령제한 드라마 수 시각화

분석 결과

  • 증가하는 그래프 확인
  • 어떤 이유인지 다른 컬럼을 통해 확인

어떤 채널에서 만들어지는지 분석하기 위해 채널별 연령제한 드라마 제작수 시각화

  • 연령제한 드라마들을 채널별로 그룹화 후 시각화

분석 결과

  • 넷플릭스가 압도적으로 많다
  • OTT영향으로 K드라마도 방송수위에 변화가 생긴 것을 확인할 수 있었다.

K드라마가 OTT의 영향을 많이 받는것으로 확인, 주로 어떤 경로로 유입이 되는지 분석

  • 연령별 미디어 유입경로 증가량 데이터 수집하여 시각화

분석 결과

  • 예상대로 OTT를 통한 유입 증가량이 많다
  • 50대 유입 증가량이 많은 특이점 확인
  • 젊은층이 아닌 50대가 많은 이유 다른 데이터로 조사

OTT 통한 미디어 유입 증가량이 많기 때문에, OTT에 접근하기 쉬운 매체 조사

  • 연령별 매체 사용률 데이터를 수집
  • 연령별 TV, 스마트폰 각각 그룹화하여 시각화

분석 결과

  • 전체적으로 TV는 감소, 스마트폰은 증가 확인
  • 5~60대 스마트폰 사용 증가를 통해 OTT유입이 많은것을 확인할 수 있었다
  • 스마트폰 OTT 연관 기사: https://news.nate.com/view/20231228n26441

K드라마 관심도 확인을 위해 K드라마 검색량 조사

  • 구글에서 K드라마 검색량 데이터 수집
  • 연도별 K드라마 검색량 시각화

분석 결과

  • 2023-04-09 이상치 확인
  • 조사를 통해 누누티비 종료 시점인 것을 확인
  • K드라마에 인기와 관심이 큰 만큼 좋지 않은 방법으로도 접근하려는 경우가 증가
  • 누누티비 관련 기사: https://www.inews24.com/view/1585549
